Проект "Сайт для изучения аглийского языка"

Сделан 2 мая 2023 года Григорием Фетисовым

База данных

Данный проект строился в основном на шаблоне "Product" с сайта bootstrap. Однако вносились значительные изменения в содержание и логику работы проекта. Использовались некоторые точечные реализации проекта с лекций и семинаров.

База данных по проекту находится в файле words.db. Состоит из двух таблиц: words и lessons. Таблицы синхронизированы с соответствующими таблицами на страницах сайта. То есть при добавлении и удалении элемента элемент также удаляется из таблицы в базе данных

Модели ORM, соответствующие таблицам, лежат в файле proj-maths/models.py

Запуск приложения

Для запуска приложения требуется ввести следующую команду в терминале

python manage.py runserver

Расположение файлов

  • HTML шаблоны располагаются в папке templates
  • Статические элементы в папке static
  • Взаимодействие с базой данных построено в файле proj_maths/terms_work.py и использует ORM
  • Связь интерфейса и логики работы проекта реализована в файле proj_maths/views.py

База данных была создана с помощью sqlite
